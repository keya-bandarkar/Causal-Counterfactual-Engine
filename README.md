# Causal Counterfactual Explanation Engine

A production-grade Counterfactual Explanation Engine for high-stakes decisions in the credit lending domain.

## Architecture

```
User Input -> [Streamlit UI] -> [Model Wrapper (XGBoost)]
                      |
                      v
             [Counterfactual Engine]
             /                     \
       [MILP Generator]        [DPP Selector]
            |                        |
[Structural Equation Model]   [Feature Bounds]
```

## Setup Instructions

1. **Install dependencies:**
   `pip install -r requirements.txt`
   
2. **Setup Data:**
   `python -m data.preprocess`
   *(This uses the default synthetic provider if Kaggle dataset isn't fully set up)*

3. **Train Model:**
   `python -m models.train`
   
4. **Discover Causal Structure:**
   `python -m causal.scm_estimator`

5. **Run Baseline vs System Evaluation benchmark:**
   `python -m evaluation.benchmark`

6. **Launch User Interface Demo:**
   `streamlit run demo/app.py`
